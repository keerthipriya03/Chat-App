# Chat-App 💬

A full-stack real-time chat application built with the MERN stack. The application enables users to communicate instantly through one-to-one and group conversations using Socket.IO. User data is securely stored in MongoDB, and authentication is implemented to provide a secure chatting experience.

## 🚀 Live Demo

[Live Application](chat-app-a30b.onrender.com)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Context API

### Backend

* Node.js
* Express.js
* Socket.IO

### Database

* MongoDB
* Mongoose

### Authentication & Security

* JWT (JSON Web Tokens)
* Encrypted user data storage

---

## ✨ Features

* 🔐 User Authentication (Login & Signup)
* 💬 Real-Time Messaging with Socket.IO
* 👤 One-to-One Chat
* 👥 Group Chats
* 🔍 Search Users
* ➕ Add Users to Groups
* ➖ Remove Users from Groups
* 📝 Update Group Information
* 👀 View User Profiles
* 📱 Responsive User Interface

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/keerthipriya03/chat-app.git
cd Chat-App
```

### Install Dependencies

```bash
npm install
npm install --prefix frontend
```

### Configure Environment Variables

Create a `.env` file inside the `Backend` directory and add the variables available in the .env.example file


### Run the Application

#### Start Backend Server

```bash
npm start
```

#### Start Frontend

Open a new terminal:

```bash
cd frontend
npm start
```
